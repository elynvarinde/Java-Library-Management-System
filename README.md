To run it:

```bash
javac LibraryManagementSystem.java
java LibraryManagementSystem
```

**What's included:**

**Books** — add, edit, remove, list, and search by title/author/ISBN/genre, with copy tracking.

**Members** — register, edit, deactivate, with a borrow limit of 5 books per member.

**Loans** — borrow and return with automatic 14-day due dates, overdue detection, and fine calculation ($0.50/day).

**Reports** — library summary, most borrowed books, most active members, fines breakdown, and genre statistics.

**Preloaded sample data** — 12 books across genres (Fiction, Sci-Fi, Tech, Self-Help, etc.) and 5 members with 3 active loans, so you can explore immediately without entering data manually.
